PRORAMMING ACTIVITIES AND COURSE WORK (BAGONZA IMRAN MIQDAD)
A collection of high-level Java logic modules designed for systems architecture and robotics. This project focuses on Object-Oriented Programming (OOP) and efficient decision-making algorithms.

Key Features
Modular Logic: Independent classes for sensors, messaging, and data management.

Scalable Architecture: Built to be easily "bridged" with other languages like Python for robotics applications.

Constraint Management: Implemented logic for character limits (280-char) and real estate price comparison.

Technical Stack
Language: Java 17+

Environment: NetBeans / IntelliJ IDEA

Concepts: Encapsulation, ArrayLists, and Conditional Logic.

"I am building these modules not just as exercises, but as the foundation for a future software ecosystem.
My goal is to bridge complex logic with physical hardware (Robotics)."

HOW TO RUN
1. Clone the repo. 2. Open in NetBeans. 3. Run Main.java.
